Point groups are the set of symmetry elements that a molecule possesses.

| Group name          | Definition                                                                                                                                          | Example                                          |
|---------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------|
| \\(C\_{2v}\\)       | \\(C\_2\\), two \\(\sigma\_v\\)                                                                                                                     | Water, **ortho**-dichlorobenzene                 |
| \\(C\_{2h}\\)       | \\(C\_2\\), \\(\sigma\_h\\), \\(i\\)                                                                                                                | **trans**-1,2-dichloroethene                     |
| \\(C\_{3v}\\)       | \\(C\_3\\), three \\(\sigma\_v\\)                                                                                                                   | Ammonia, chloroform                              |
| \\(D\_{2h}\\)       | Three perpendicular \\(C\_2\\) axes, \\(\sigma\_h\\)                                                                                                | **para**-dichlorobenzene, ethylene, naphthalene  |
| \\(D\_{3h}\\)       | Principal \\(C\_3\\), 3x \\(C\_2\\) perpendicular to principal axis, 3x \\(\sigma\_v\\), \\(\sigma\_h\\), \\(S\_3\\) coinciding with principal axis | Trigonal planar molecules like boron trichloride |
| \\(T\_d\\)          | Symmetries present in tetrahedral molecule with identical substituents around central atom                                                          | Methane                                          |
| \\(O\_h\\)          | Symmetries present in octahedral molecules with identical substituents around central atom                                                          | Sulfur hexafluoride                              |
| \\(C\_{\infty v}\\) | \\(C\_\infty\\) axis, infinite number of \\(\sigma\_v\\)                                                                                            | Heteronuclear diatomics, SCO                     |
| \\(D\_{\infty h}\\) | \\(C\_{\infty v}\\) but with infinite number of \\(C\_2\\) perpendicular to \\(C\_\infty\\) axis                                                    | Homonuclear diatomics, carbon dioxide            |
